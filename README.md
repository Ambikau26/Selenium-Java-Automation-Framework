Automation Testing Framework - Selenium Java

Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model (POM)

Project Description
This project is a Selenium Automation Testing framework developed using Java and TestNG. It automates end-to-end test scenarios for web applications including login, product selection, and checkout workflows.

Key Features
- Page Object Model design pattern
- TestNG test execution
- Maven dependency management
- Automated test reports generation

How to Run
1. Clone repository
2. Import project as Maven project in Eclipse/IntelliJ
3. Run testng.xml file
